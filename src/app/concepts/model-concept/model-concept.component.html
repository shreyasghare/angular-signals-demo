model()

Two-way binding signal for parent-child component communication

Overview

The model() function creates a writable signal that can be used for two-way binding between parent and child components. It combines the functionality of input and output signals.

Usage

import {{ '{' }} model {{ '}' }} from '@angular/core';

// In parent component
sharedValue = model('');

// In child component template
<app-child [(sharedModel)]="sharedValue"></app-child>

// In child component
sharedModel = model.required<string>();

Key Features

  • Two-way binding - combines input and output in one signal
  • Writable - can be updated from both parent and child
  • Reactive - changes propagate automatically
  • Type-safe - TypeScript types are enforced
