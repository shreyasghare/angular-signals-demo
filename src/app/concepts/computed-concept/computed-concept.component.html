computed()

Derived reactive values that automatically update when dependencies change

Overview

The computed() function creates a read-only signal that derives its value from other signals. It automatically tracks dependencies and updates when they change.

Usage

import {{ '{' }} signal, computed {{ '}' }} from '@angular/core';

firstName = signal('John');
lastName = signal('Doe');

// Create a computed signal
fullName = computed(() => {{ '{' }}
  return `${{ '{' }}this.firstName(){{ '}' }} ${{ '{' }}this.lastName(){{ '}' }}`;
{{ '}' }});

// Read the computed value
const name = fullName();

Key Features

  • Read-only - cannot be directly updated with set()
  • Automatic dependency tracking - tracks which signals it depends on
  • Lazy evaluation - only recomputes when dependencies change
  • Memoized - caches the result until dependencies change
