effect()

Side effects that run when signals change

Overview

The effect() function creates a side effect that runs when its signal dependencies change. Effects automatically track which signals they read.

Usage

import {{ '{' }} signal, effect {{ '}' }} from '@angular/core';

count = signal(0);

// Create an effect
effect(() => {{ '{' }}
  console.log('Count changed:', this.count());
{{ '}' }});

// Effects run automatically when dependencies change
count.set(5); // Effect runs and logs: "Count changed: 5"

Key Features

  • Automatic dependency tracking - tracks signals read inside the effect
  • Runs synchronously - executes immediately when dependencies change
  • Can be used for side effects like logging, DOM updates, or API calls
  • Returns a cleanup function - can be used to clean up resources
