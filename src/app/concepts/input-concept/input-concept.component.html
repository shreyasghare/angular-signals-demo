input()

Component input signal for receiving data from parent components

Overview

The input() function creates a read-only signal that receives values from a parent component. It replaces the traditional @Input() decorator with a signal-based approach.

Usage

import {{ '{' }} input {{ '}' }} from '@angular/core';

// In child component
userName = input.required<string>();
age = input(0); // Optional with default value

// Use in template
<p>Hello, {{ '{' }}{{ '{' }} userName() {{ '}' }}{{ '}' }}!</p>

// In parent component template
<app-child [userName]="fullName()" [age]="25"></app-child>

Key Features

  • Read-only - cannot be updated from the child component
  • Reactive - automatically updates when parent changes the value
  • Type-safe - TypeScript types are enforced
  • Can be required or optional with default values
