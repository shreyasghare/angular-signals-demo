signal()

Basic reactive state primitive for creating writable signals

Overview

The signal() function creates a writable signal that holds a value. Signals are reactive primitives that can be read and updated.

Usage

import {{ '{' }} signal {{ '}' }} from '@angular/core';

// Create a signal with initial value
count = signal(0);

// Read the signal value
const currentCount = count();

// Update the signal
count.set(10);

// Update using a function
count.update(value => value + 1);

Key Features

  • Writable - can be updated with set() or update()
  • Reactive - changes automatically trigger updates in dependent computed signals and effects
  • Type-safe - TypeScript types are inferred from the initial value
  • Readable - call the signal as a function to read its value

Example in Demo

In the Signals Demo, basic signals are used for form fields like firstName, lastName, age, and email. These signals are updated as the user types in the input fields.
