output()

Component output event signal for emitting events to parent components

Overview

The output() function creates an output signal that allows child components to emit events to parent components. It replaces the traditional @Output() decorator with a signal-based approach.

Usage

import {{ '{' }} output {{ '}' }} from '@angular/core';

// In child component
userSubmitted = output<UserData>();

// Emit an event
this.userSubmitted.emit({{ '{' }}
  userName: 'John Doe',
  timestamp: Date.now()
{{ '}' }});

// In parent component template
<app-child (userSubmitted)="onUserSubmitted($event)"></app-child>

// In parent component
onUserSubmitted(event: UserData) {{ '{' }}
  console.log('Received:', event);
{{ '}' }}

Key Features

  • Event emission - child components can emit events to parents
  • Type-safe - TypeScript types are enforced for event data
  • Reactive - can be used with signals and effects
  • Replaces @Output() decorator with signal-based API
